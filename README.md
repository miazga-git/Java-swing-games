# Java Swing games

In this repository there are 5 folders, each one contains game package. Two games are from YouTube tutorial - T-Rex Game and Snake Game. Other three games I' ve made on my own - Minesweeper, BallBouncing Game, Brick Breaker Game

### How to play

To play the game use mouse. Left mouse click the squares to uncover them. Right mouse click the squares to flag or mark them. To restart the game click on the emoji icon.

## BallBouncing Game
<p align="center">
<img src="https://user-images.githubusercontent.com/82395921/219115059-2fad51bc-ca90-426e-aa03-2b78f8e63370.png" width="800" height="500">
</p>

### How to play

<kbd>w</kbd> - player 1 platform goes up

<kbd>s</kbd> - player 1 platform goes down

<kbd>up</kbd> - player 2 platform goes up

<kbd>down</kbd> - player 2 platform goes down

<kbd>space</kbd> - restart

### How to play

<kbd>Up arrow</kbd> - snake starts to crawling up

<kbd>Down arrow</kbd> - snake starts to crawling down

<kbd>Left arrow</kbd> - snake starts to crawling left

<kbd>Right arrow</kbd> - snake starts to crawling right

<kbd>Space</kbd> - restart
